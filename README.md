PiPurr for Android
==================

PiPurr client for Android.

See PiPurr Server repo for the server.

Features:
* Get static image of cats.
* Play meow sound through speakers on server.
* Share image via sharing intent.

Coming soon:
* optimise sharing.
* video feed.
* cat interaction.
